二分搜索模板

题目

给一个有序数组和目标值，找第一次/最后一次/任何一次出现的索引，如果没有出现返回-1

通用模板
四点要素:

  1. start + 1 < end
  2. start + (end - start) / 2
  3. A[mid] ==, <, >
  4. A[start] A[end] ? target

大部分二分查找类的题目都可以用这个模板，然后做一点特殊逻辑即可

常见题目：

  1. 二分查找

  2. 矩阵搜索

  3. 旋转数组搜索

核心思路：

根据nums[start]<nums[mid] 判断两种情况

总结

  • 每一个模板都是前人总结的经验。
  • 没有模板就自己创造一个模板。
