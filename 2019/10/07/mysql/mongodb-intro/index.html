mongodb

概念

优势

  • 格式不固定 JSON
  • MapReduce 复杂的聚合查询
  • 支持全文索引
  • 支持分片分布式存储

缺点

  • 索引需要手动创建
  • 索引在内存创建（内存不够，则不会创建索引，或者删除部分索引）
